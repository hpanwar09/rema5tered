# what should i watch?

movies, puzzles, and my letterboxd diary — all in one place.

## what's here

- **what should i watch?** — tell me your mood, i'll find you a movie
- **guess the movie** — pixelated poster, five guesses, same for everyone daily
- **my recent watches** — pulled live from my letterboxd
- **about me** — who i am, why i built this

## tech

astro, react, tailwind, vercel. movie data from tmdb. watch history from letterboxd rss.

## run locally

```
cp .env.example .env
# add your tmdb api key from themoviedb.org/settings/api
npm install
npm run dev
```
